Internet


HTML - Hypertext Markup Language. É uma linguagem de descrição de páginas de informação, standard no WWW. Com esta linguagem (que, para além do texto, tem comandos para introdução de imagens, formulários, alteração de fontes, ect.) podem-se definir páginas que contenham informação nos mais variados formatos:
  • texto
  • som
  • imagens
  • animações.
